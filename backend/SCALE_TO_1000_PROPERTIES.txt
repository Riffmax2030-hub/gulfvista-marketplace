================================================================================
TROUBLESHOOTING
================================================================================

Problem: Image script takes very long (>10 minutes)
Solution:
- This is normal for 82 properties over internet connection
- You can interrupt and run again: Ctrl+C
- Images will still be saved

Problem: "Connection error" when fetching images
Solution:
- Check your internet connection
- Try again later if Unsplash API is temporarily down
- Images will use placeholder if API fails

Problem: Only showing 82 properties, not 1000+
Solution:
1. Verify bulk generation script ran: "Properties generated: 920"
2. Hard refresh browser multiple times: Ctrl+F5
3. Check pagination - should show 80+ pages
4. Wait 5 seconds for database to sync

Problem: Duplicate neighborhoods appearing
Solution:
- This is normal - same neighborhood in different cities
- Example: "Downtown Dubai" vs "Downtown Doha" are different
- Shows realistic global presence

Problem: Properties loading slowly
Solution:
- 1000+ properties is a lot to display
- Pagination to 12 per page helps performance
- Consider adding filters (by price, location, etc.)
